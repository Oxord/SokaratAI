# UI-дизайн: $ARGUMENTS

## Основные принципы

Перед написанием кода определи чёткое эстетическое направление. Обдумай:
- Назначение интерфейса
- Тон и характер
- Ограничения платформы (iOS UIKit / web)
- Что делает этот интерфейс особенным

Ключевое правило: «Выбери чёткое концептуальное направление и реализуй его с точностью. Смелый максимализм и утончённый минимализм одинаково работают — важна намеренность, а не интенсивность.»

## Чего избегать (типичная AI-эстетика):

- Системные шрифты по умолчанию без характера
- Безопасные предсказуемые синие градиенты
- Симметричные, сеточно-зафиксированные макеты без вариаций
- Безликий card-based дизайн
- Шаблонные паттерны без контекстного характера

## Чеклист дизайна:

1. **Типографика**: Используй выразительные шрифты с характером. Для iOS — SF Pro с тщательным подбором весов и размеров. Определи чёткую иерархию.
2. **Цветовая палитра**: Цельная схема через UIColor extensions. Один доминирующий цвет + острый акцент. Учитывай фитнес-домен: энергичные, мотивирующие тона.
3. **Анимации и движение**: Одна хорошо выстроенная загрузка страницы со staggered reveals лучше множества разрозненных анимаций. UIView.animate со spring timing. Соблюдай reduceMotion.
4. **Пространственная композиция**: Используй асимметрию, наложения, неожиданные макеты там, где уместно. Не скатывайся в скучные сетки.
5. **Визуальные детали**: Атмосферные фоны, тонкие текстуры, тени с назначением — всё согласованно с эстетическим видением.
6. **Доступность**: Контраст WCAG AA, Dynamic Type, метки VoiceOver.

## Рабочий процесс:

1. Проанализируй контекст ($ARGUMENTS) — какой интерфейс строится
2. Определи эстетическое направление (кратко, 2-3 предложения)
3. Реализуй с педантичным вниманием к деталям
4. Код должен быть production-grade И визуально выразительным
5. Сопоставь сложность с видением: максимализм = сложные эффекты, минимализм = точность в отступах и типографике

## Специфика платформ:

- **iOS UIKit**: UIColor, UIFont, NSLayoutConstraint, UIView.animate, CALayer effects
- **Web**: CSS custom properties, градиенты, анимации, responsive design
