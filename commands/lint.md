Запусти линтер и форматтер, исправь найденные проблемы.

## Действия

1. Запусти проверку: `.venv/bin/ruff check src tests`
2. Запусти форматирование: `.venv/bin/ruff format src tests`
3. Если ruff check нашёл авто-исправимые ошибки — запусти `.venv/bin/ruff check --fix src tests`
4. Если остались ошибки, которые нельзя автоисправить — покажи каждую с объяснением и предложи фикс
5. Кратко сообщи итог: сколько файлов проверено, сколько исправлено
